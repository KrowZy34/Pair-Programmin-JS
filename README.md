# Pair-Programmin-JS
Pyoecto simple de pair programming en JavaScript


-Sistema de restaurante 
En el archivo menu.js se encuentra toda la data que vamos a utlizar para el proyecto
con dos funciones en general

En el archivo operaciones.js se encuentra toda las operaciones que vamos realizar en el proyecto

En el archivo ui.js se encuentra la logica pero la interfaz para que funciones los botones y muestra la data en el html.

En el archivo main.js se conencta todo para que funcione el proyecto.

En el archivo index.htm se encuentra la interfaz del proyecto.

Porque es un estandar basico de la modulacion ideal para practicar.
